

# Search Trends Analysis (2001–2020)

## Overview

This project performs **Exploratory Data Analysis (EDA)** on a search trends dataset to understand patterns across **years, categories, locations, and rankings**.
The analysis focuses on identifying popularity trends, top categories, and ranking distribution using Python visualization libraries.

---

## Dataset Information

The dataset contains **26,955 records** with the following columns:

| Column   | Description                             |
| -------- | --------------------------------------- |
| location | Geographic location of the search trend |
| year     | Year of the trend (2001–2020)           |
| category | Category of the search query            |
| rank     | Popularity rank (1 to 5)                |
| query    | Search query text                       |

---

## Objectives

* Analyze trends over time (2001–2020)
* Identify top-performing categories
* Study rank distribution
* Explore category popularity across years
* Visualize insights for better understanding

---

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Visualizations Included

* Year-wise trend analysis
* Rank distribution
* Top categories
* Top locations
* Category popularity over time (Heatmap)
* Average rank trends

---

## Sample Insight

* Search data shows increasing activity in recent years.
* A few categories dominate overall popularity.
* Rank distribution indicates consistent competition among top trends.
